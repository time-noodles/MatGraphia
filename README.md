# MatGraphia

固体化学・材料科学研究室向けの実験管理システムです。合成イベント・サンプル・測定データ・文献を一元管理し、XRD シミュレーションや結晶構造可視化、Obsidian ノートへの自動エクスポートを提供します。

## 概要

MatGraphia は以下の研究ワークフローをサポートします。

```
物質 (CIF) ──→ イベント (合成) ──→ サンプル ──→ 測定 (XRD / SEM)
                                                      ↓
                                              Obsidian ノート出力
```

主な機能:
- **文献管理**: 論文・学内論文の DOI・PDF 付き登録
- **物質情報管理**: CIF ファイルによる結晶構造登録・XRD シミュレーション・3D 可視化
- **イベント管理**: 固相合成 / CVT / アルゴンアニール / 脱インターカレーションなどの合成手順記録
- **サンプル管理**: イベントから生成したサンプルの追跡
- **測定データ管理**: XRD・SEM 測定データの登録と自動解析
- **Obsidian 連携**: 全データを Markdown + 画像として自動エクスポート

## 動作環境・インストール

### 必要要件

- Python 3.10 以上
- Git（任意）

### インストール

```bash
# リポジトリのクローン（またはフォルダをコピー）
cd MatGraphia

# 依存ライブラリのインストール
pip install -r requirements.txt
```

主要な依存ライブラリ:

| ライブラリ | 用途 |
|---|---|
| streamlit | Web UI |
| pymatgen | XRD シミュレーション・結晶構造解析 |
| crystal-toolkit | 3D 結晶構造可視化 |
| pydantic | データバリデーション |
| pdfplumber | SEM レポート PDF 解析 |
| pandas / numpy | データ処理 |
| matplotlib | XRD プロット生成 |

---

## 起動方法

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

初回起動時に `data/` ディレクトリと SQLite データベース (`data/matgraphia.db`) が自動作成されます。

## データモデル

6 種類のエンティティを管理します。各エンティティは UUID で識別されます。

```
Literature (文献)
     ↑ 参照
Event (イベント) ─── 入力サンプル参照 ───→ Sample (サンプル)
     ↓ 生成                                    ↓ 測定
  Sample ──────────────────────────────→ Measurement (測定)
                                              ↑ 比較
Material (物質) ──────────────────────→ ComparisonFigure (比較図)
```

### エンティティ一覧

| エンティティ | 説明 |
|---|---|
| **Literature** | 参考文献（論文・学内論文）|
| **Material** | 物質情報（CIF ファイル・物性値）|
| **Event** | 合成実験の手順・条件 |
| **Sample** | イベントから得られたサンプル |
| **Measurement** | サンプルの測定データ（XRD・SEM）|
| **ComparisonFigure** | 複数測定の比較図 |

## 各ページの使い方

### 文献の登録

**サイドバー →「文献の登録」**

1. **文献タイプを選択**: 「Paper」または「学内論文」
2. **必須項目を入力**:
   - DOI（必須）
   - タイトル・著者・掲載誌・出版年・巻号
   - 備考
3. **PDF をアップロード**（任意）: `data/Literature/<id>/` に保存されます
4. **「登録」ボタン**をクリック

学内論文の場合は学位種別（学士・修士・博士）、氏名、年度を追加入力します。

---

### 物質情報の登録

**サイドバー →「物質情報の登録」**

1. **物質名を入力**: 化学式（例: `NaCrS2`）
2. **CIF ファイルをアップロード**（任意）: `data/Material/<id>/` に保存
   - アップロード後、自動で XRD シミュレーション結果が `properties` に格納されます
   - 「結晶構造プレビュー」ボタンで 3D 可視化（別ブラウザウィンドウで Dash アプリが起動）
3. **物性値を入力**（任意）: Tc、格子定数などを JSON 互換形式で記録
4. **関連文献を選択**（任意）
5. **不純物相・多形を指定**（任意）: 他の Material レコードを選択
6. **「登録」ボタン**をクリック

#### CIF アップロード後の XRD シミュレーション

CIF ファイルがある物質は、測定 XRD との比較時に自動でシミュレーションパターンを表示します。「シミュレーション条件」に合わせて測定条件（線源・角度範囲）を設定しておくと、比較図の精度が上がります。

---

### イベントの登録

**サイドバー →「イベントの登録」**

イベント = 合成実験 1 回分の記録です。

1. **プロジェクト名を入力**
2. **目的物質を入力**（例: `NaCrS2`）
3. **イベントタイプを選択**:

| タイプ | 説明 |
|---|---|
| Solid-State | 固相焼成合成 |
| CVT | 化学気相輸送法 |
| Ar-Annealing | アルゴン雰囲気アニール |
| Deintercalation | 脱インターカレーション |

4. **タイプ固有のパラメータを入力**:
   - **Solid-State / CVT / Ar-Annealing**: 管炉情報、石英管サイズ、原料仕込み表、昇温プロファイル
   - **Deintercalation**: 温度・時間・溶媒・酸化剤・攪拌・サンプル形態

5. **入力サンプルを選択**（任意）: このイベントで消費したサンプル
6. **参照イベント・参照文献を選択**（任意）
7. **「登録」ボタン**をクリック

#### 昇温プロファイルの入力例（Solid-State）

| Heating Temperature (°C) | Duration (h) | Target End |
|---|---|---|
| 800 | 12 | False |
| 1000 | 48 | True |
| 25 | 6 | False |

「Target End」にチェックを入れた行が最終保持温度として扱われます。

---

### サンプルの登録

**サイドバー →「サンプルの登録」**

サンプル = イベントから得られた実物の材料です。

1. **ソースイベントを選択**（必須）
2. **サンプル ID（ラボ内通称）を入力**（例: `01-NaCrS2-1st`）
3. **サンプル形態を選択**: 現在「Powder（粉末）」のみ
4. **保管場所を入力**（任意）
5. **備考を入力**（任意）
6. **「登録」ボタン**をクリック

---

### 測定データの登録

**サイドバー →「測定データの登録」**

#### XRD 測定

1. **測定タイプ**で「XRD」を選択
2. **測定サンプルを選択**
3. **測定条件を入力**:
   - 線源: CuKα / MoKα / CoKα / FeKα
   - 2θ 範囲・ステップ幅・ピーク幅
   - シミュレーションモード: Powder（粉末）/ TMDC c-axis Oriented（c 軸配向）
4. **生データをアップロード**: Bruker RAW 4.00 形式または CSV/TXT
5. **「解析・登録」ボタン**:
   - ピーク位置の自動抽出
   - 対応物質の XRD シミュレーションとの比較図を自動生成
   - 比較図（PNG）が `data/Processed/<id>/` に保存

#### SEM 測定

1. **測定タイプ**で「SEM」を選択
2. **PDFレポートをアップロード**: JSM-IT700HR の出力 PDF から元素組成（At%）を自動抽出
3. **加速電圧・装置名を確認**（自動入力）
4. **元素組成テーブルを確認・修正**
5. **「登録」ボタン**をクリック

---

### データ管理・編集

**サイドバー →「データ管理・編集」**

登録済みの全レコードを一覧表示・編集・削除できます。

- 各テーブルで行を選択して編集フォームを表示
- 削除時は関連ファイルも削除されます
- XRD 比較図の再生成も可能

## Obsidian 連携

アプリ右側の「Obsidian Sync」パネルから操作します。

### 自動同期

- **「自動同期を有効にする」トグル**: データ変更のたびに `obsidian_vault/` を自動更新

### 手動エクスポート

- **「今すぐエクスポート」ボタン**: 全データを `obsidian_vault/` に書き出し
- **「ZIP でダウンロード」ボタン**: Vault 全体を ZIP ファイルとして取得

### エクスポートされるファイル構成

```
obsidian_vault/
├── Literature/          # 文献ノート
├── Event/               # イベントノート
├── Sample/              # サンプルノート
├── Measurement/         # 測定ノートと XRD プロット埋め込み
├── Material/            # 物質ノート
└── _assets/             # 埋め込み XRD 画像 (PNG)
```

各ノートは YAML フロントマターとバックリンク（`[[]]`）形式で出力され、Obsidian 上でエンティティ間のグラフが構築されます。

## プラグインシステム

`plugins/` ディレクトリに Python ファイルを置くことで、イベントタイプ・測定タイプ・文献タイプなどを追加できます。

### プラグインの構造

各プラグインファイルは `SCHEMA` 辞書をエクスポートします。

```python
# plugins/events/my_method.py

SCHEMA = {
    "type": "My-Method",          # タイプ識別子（一意）
    "fields": [
        {"name": "temperature",   "type": "float",   "default": 25.0},
        {"name": "atmosphere",    "type": "str",     "default": "N2"},
        {"name": "mode",          "type": "select",  "options": ["A", "B"], "default": "A"},
    ],
    "tables": [
        {
            "name": "my_table",
            "columns": [
                {"name": "step",  "type": "int"},
                {"name": "value", "type": "float"},
            ]
        }
    ]
}
```

### フィールド型

| `type` | Streamlit ウィジェット |
|---|---|
| `str` | テキスト入力 |
| `int` | 整数入力 |
| `float` | 小数入力 |
| `bool` | チェックボックス |
| `select` | セレクトボックス（`options` 必須）|

### プラグインカテゴリ

| ディレクトリ | 対象 |
|---|---|
| `plugins/events/` | 合成イベントタイプ |
| `plugins/samples/` | サンプル形態 |
| `plugins/measurements/` | 測定タイプ |
| `plugins/literatures/` | 文献タイプ |
| `plugins/materials/` | 物質可視化ツール |
| `plugins/visualizations/` | カスタム可視化 |

## ディレクトリ構成

```

├── app.py                  # Streamlit エントリポイント
├── database.py             # SQLite データベース操作

├── plugin_manager.py       # プラグイン動的ローダー
├── obsidian_exporter.py    # Obsidian Markdown エクスポート
├── schemas.py              # Pydantic データモデル
├── requirements.txt        # 依存ライブラリ一覧
├── plugins/
│   ├── events/             # 合成イベントプラグイン
│   │   ├── solid_state.py
│   │   ├── cvt.py
│   │   ├── ar_annealing.py
│   │   └── deintercalation.py
│   ├── literatures/        # 文献タイププラグイン
│   │   ├── paper.py
│   │   └── internal_thesis.py
│   ├── samples/            # サンプル形態プラグイン
│   │   └── powder.py
│   ├── measurements/       # 測定タイププラグイン
│   │   ├── xrd.py
│   │   └── sem.py
│   └── materials/          # 物質可視化プラグイン
│       └── structure_preview.py
├── data/                   # データ保存先（自動生成）
│   ├── matgraphia.db       # SQLite データベース
│   ├── Literature/         # 文献 PDF
│   ├── Material/           # CIF ファイル
│   ├── Raw/                # 生測定データ
│   └── Processed/          # 処理済みデータ・XRD 比較図
└── obsidian_vault/         # Obsidian エクスポート先（自動生成）
```
